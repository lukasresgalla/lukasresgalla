### Principais Projetos

**<a href="https://github.com/lukasresgalla/park-share" target="_blank">Park-Share</a>** - Um sistema de gerenciamento de aluguel de vagas de garagem. Construído com Python e PostgreSQL. O projeto é conteinerizado com Docker e segue uma arquitetura robusta separando modelos, DAOs, repositórios e controladores.<br>

**<a href="https://github.com/lukasresgalla/alarm-system" target="_blank">Alarm-System</a>** - Demonstração de uma arquitetura de microsserviços em Python. O sistema utiliza um API Gateway (Flask) para orquestrar a comunicação via REST APIs entre múltiplos serviços independentes (ex: serviço de usuários, serviço de alarmes, serviço de logging).<br>

**<a href="https://github.com/lukasresgalla/gerenciamento-memoria-paginacao" target="_blank">Simulador de Gerenciamento de Memória Paginada</a>** - Um simulador de gerenciamento de memória com paginação escrito puramente em **C**. Este projeto acadêmico implementa conceitos fundamentais de Sistemas Operacionais, incluindo tabelas de páginas, alocação de quadros (frames) e gerenciamento de processos em um menu interativo.<br>

**<a href="https://github.com/lukasresgalla/FoodPark" target="_blank">FoodPark</a>** - Um aplicativo mobile simples para um food park, desenvolvido em **React Native**. Permite ao usuário visualizar lojas, cardápios, eventos e gerenciar uma cesta de compras.<br><br>
